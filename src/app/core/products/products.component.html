Wireless Light Control is an optional feature of Fee Machine.

When enabled, Fee Machine will turn on a light via Wifi above your Billiard Table or Lane at the start of each rental, and turn it off at the end, providing an easy way to monitor which items are being actively rented.

With Wireless Light Control you have the ability to set light individual bulb color settings to either Cool or Warm.

Alternately, Lights can be configured to turn on Bright White when a Prepaid Rental starts, then Yellow when the Rental is nearing it's end, and finally Flash Red when the Rental Time has expired.

Wireless Light Control requires a wireless network with all light bulbs and your Fee Machine tablet attached to that network. Contact your Fee Machine reseller for more details.

Fee Machine makes it easy to find and configure your wireless lighting controls with your rentals!

Aloha POS

Fee Machine is designed work with Aloha Table Service or Quick Service system without the need for additional licensing from NCR!


Using Barcode Scanner Technology, Fee Machine works with any Aloha Terminal that can be equipped with a 2D Bar Code scanner.


Multiple Rental Fees can be sent to your Aloha system with a single scan, each of which can have its own item id and price.


No need for Aloha software upgrades or additional licenses from NCR!

Contact your Aloha Reseller for more information.
